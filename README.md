### Candlestick chart
- 캔들스틱 하나는 일간, 월간, 연간 집계된 통계 누적치 될 수 있다
- 스틱 안에는 최댓값, 최솟값, 평균, 중간값 등 다양한 통계 누적 가능
- 상위25%=upper shadow, 하위25%=lower shadow
